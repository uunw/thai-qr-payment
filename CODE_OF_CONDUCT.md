# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, focus discussion on the code. Harassment, personal attacks, and discriminatory remarks are not tolerated.

Issues can be reported privately via [GitHub Private Vulnerability Reporting](https://github.com/uunw/thai-qr-payment/security/advisories/new) (used for both security and conduct reports). The maintainer will respond within 7 days and act in proportion to the report.
